# Documentation Directory

This directory contains comprehensive documentation for the Materials Demand Model.

## Documentation Files

### Core Documentation (CSV Format)

These CSV files provide structured, machine-readable documentation of the pipeline:

#### **`variable_reference.csv`** (135 entries)
**Purpose:** Documents all pipeline variables, parameters, features, and calculations

**When to use:** Look up variable definitions, calculation formulas, or units for any quantity used in the simulation, feature engineering, clustering, or sensitivity analysis.

**Contents:**
- Input data variables (intensity, technology, material, scenario, year, capacity)
- Configuration parameters (technology mapping, lifetimes, weights)
- Stock-flow model variables (stock, additions, retirements)
- Monte Carlo outputs (demand statistics, percentiles)
- Engineered features (17 scenario features, 23 material features)
- Risk components (supply chain risk measures)
- Preprocessing variables (log transform, standardization, VIF)
- Clustering variables (Sparse PCA/Factor Analysis scores, configs)
- Sensitivity analysis variables (variance decomposition, Spearman/Sobol metrics)

**Columns:** Variable/Parameter, Category, Units, Description, Calculation Formula, Raw Data Source(s)

---

#### **`data_sources.csv`** (9 entries)
**Purpose:** Documents all INPUT data files and their structure

**When to use:** Understand where raw data comes from, what format it's in, and what sheets/columns are available.

**Contents:**
- `intensity_data.csv` — Material intensity values
- `StdScen24_annual_national.csv` — NREL capacity projections
- `risk_charts_inputs.xlsx` — Supply chain risk data (with sheet descriptions)
- `mcs2023-*_salient.csv` — USGS material-specific CSVs

**Columns:** File Name, Type, Format, Description, Sheets/Components, Source, Location

---

#### **`output_files.csv`** (36 entries)
**Purpose:** Documents all OUTPUT files and directories generated by the pipeline

**When to use:** Find where specific analysis results are saved, what format they're in, and which script generates them.

**Contents:**
- Core outputs (`material_demand_by_scenario.csv`)
- Sensitivity analysis results (variance decomposition, elasticity, Spearman, Sobol)
- Clustering outputs (scores, loadings, communalities, comparison metrics)
- Output directories (manuscript figures, clustering, exploratory, supply chain risk)

**Columns:** File/Directory Name, Category, Format, Description, Key Columns/Contents, Generated By

---

#### **`visualization_inventory.csv`** (80 entries)
**Purpose:** Tracks all visualizations with their input variables and data sources

**When to use:** Find which figures are available, what data they use, and how to regenerate them.

**Columns:** Visualization Name, Output Path, Category, X-axis, Y-axis, Color, Additional Variables, Data Source, Script
