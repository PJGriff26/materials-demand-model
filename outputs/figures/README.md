# Figures Directory

## Paper Figures (`manuscript/`)

All figures that appear in the manuscript (main text + supplementary).
Generated by `visualizations/generate_manuscript_figures.py`.

```
manuscript/
  fig1_demand_projections.png        # Fig 1: Annual demand (12-panel grid)
  fig2_technology_decomposition.png  # Fig 2: Technology breakdown (stacked bar)
  fig3_variance_decomposition.png    # Fig 3: Intensity vs scenario variance
  fig4_material_clusters.png         # Fig 4: Material clusters SPCA biplot
  fig5a_supply_risk_matrix.png       # Fig 5a: Global supply risk scatter
  fig5b_us_supply_risk_matrix.png    # Fig 5b: US supply risk scatter
  fig5a_supply_chain_sourcing.png    # Fig 5a alt: CRC sourcing breakdown
  fig5b_reserve_adequacy.png         # Fig 5b alt: Reserve adequacy (global)
  fig5c_reserve_adequacy_us.png      # Fig 5c: Reserve adequacy (US)
  si_figures/
    si_fig_s1  ... si_fig_s21        # 21 supplementary figures
```

Regenerate: `python visualizations/generate_manuscript_figures.py`
Subset:     `python visualizations/generate_manuscript_figures.py --only fig1 fig3`
List:       `python visualizations/generate_manuscript_figures.py --list`

## Analysis Figures (not in paper)

Everything outside `manuscript/` is exploratory or intermediate analysis output.
These are produced by the pipeline (`run_pipeline.py`) or individual analysis scripts,
and are useful for development and review but do not appear in the manuscript.

- `clustering/` -- clustering analysis (elbow plots, method comparisons, dim. reduction)
  - `kmeans/` -- K-means results (biplots, silhouette, profiles, spaghetti)
  - `comparison/` -- 4-method comparison (PCA, SPCA, NMF, FA)
  - `dimensionality_reduction/` -- method comparison, feature tiers
  - `factor_analysis/` -- communalities and loadings
  - `pca_analysis/` -- PCA scree, biplots, feature importance
  - `spca_story/` -- Sparse PCA narrative visualizations
- `sensitivity/` -- sensitivity analysis (Spearman, variance decomposition, tornado)
- `exploratory/` -- EDA (correlation heatmaps, scatterplot matrices)
- `supply_chain_risk/` -- risk ranking charts, component heatmaps
- `archive/` -- historical prototypes and validation figures
